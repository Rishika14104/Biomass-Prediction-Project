# Image to Biomass Prediction Using Deep Learning

## Project Description
This project predicts pasture biomass from RGB images using a ResNet50 deep learning model. Users can upload an image through a Streamlit web application to estimate biomass values.

## Technologies Used
- Python
- PyTorch
- OpenCV
- Streamlit
- NumPy
- Pandas
- Matplotlib

## Project Files
- app.py - Streamlit application
- Biomass_Project.ipynb - Training notebook
- biomass_resnet50.pth - Trained model
- requirements.txt - Dependencies
- test.py - Testing script

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
